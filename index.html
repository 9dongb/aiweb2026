AI Meeting Workflow

회의록을 다음 업무로 바로 연결합니다.

MeetingFlow AI는 회의 전사문에서 요약, 결정사항, 담당자별 액션 아이템을 추출하고 Notion, Google Calendar, 이메일 초안까지 이어주는 후속 업무 자동화 서비스입니다.

Why it matters

회의가 끝난 뒤에도 업무가 흩어지지 않도록

회의록을 다시 읽고, 담당자와 마감일을 확인하고, 공유 메일과 일정을 따로 만드는 반복 작업을 줄입니다. 분석 결과를 검토한 뒤 바로 실행 가능한 업무 데이터로 남길 수 있습니다.

  • 요약, 주요 논의, 결정사항 자동 정리
  • 담당자, 마감일, 우선순위 기반 액션 아이템 생성
  • 캘린더 일정, Notion 초안, 후속 이메일까지 연결

Problem → Solution

단순 요약이 아니라 실행 가능한 회의 결과를 만듭니다.

MeetingFlow AI는 회의 내용을 업무 단위로 구조화해 누락되는 후속 작업과 반복적인 정리 시간을 줄이는 데 초점을 둡니다.

기존 방식

회의록은 남지만 실행 항목은 흐려집니다.

담당자, 마감일, 공유 메일, 일정 등록을 사람이 각각 처리해야 해서 후속 업무가 늦어지거나 빠질 수 있습니다.

MeetingFlow AI

회의 내용을 업무 파이프라인으로 바꿉니다.

AI 분석 결과를 액션 아이템, 일정, 문서, 이메일 초안으로 이어 회의 이후의 행동까지 한 화면에서 관리합니다.

Workflow

입력부터 후속 처리까지 한 번에 이어지는 흐름

01

회의록 입력

텍스트를 붙여넣거나 .txt, .docx 파일을 업로드하고 회의 제목, 날짜, 참석자 정보를 더합니다.

02

AI 구조화

GPT API가 요약, 주요 논의, 결정사항, 미해결 이슈, 액션 아이템을 JSON 형태로 추출합니다.

03

업무 실행

대시보드와 칸반 보드에서 검토한 뒤 캘린더, Notion, 후속 이메일 초안으로 연결합니다.

Core Features

회의 후 바로 필요한 기능만 선명하게 묶었습니다.

요

회의 내용 구조화

회의 제목, 날짜, 참석자, 요약, 주요 논의 내용, 결정사항을 자동으로 정리합니다.

액

액션 아이템 관리

담당자, 마감일, 우선순위, 상태를 포함한 업무 카드를 목록 또는 칸반 형태로 확인합니다.

연

외부 도구 연동

Google Calendar 일정 동기화, Notion 회의록 초안 생성, 후속 이메일 초안 작성을 지원합니다.

팀

팀 워크스페이스

이메일 로그인과 Google OAuth를 바탕으로 팀 단위 회의와 액션 아이템을 공유합니다.

Use Cases

팀 프로젝트부터 업무 회의까지

회의가 끝난 직후 필요한 정리, 공유, 일정 등록, 기록 보관 흐름을 한 서비스 안에서 처리합니다.

회의 직후 정리 원문 붙여넣기 → AI 분석 → 결과 검토
일정 관리 액션 아이템 확인 → Google Calendar 동기화
팀 공유 분석 결과 확인 → 후속 이메일 초안 생성
기록 보관 회의 요약과 결정사항 → Notion 초안 생성

Tech Stack

AI 분석과 업무 연동을 위한 구성

Frontend Next.js 15 · React 19 · TypeScript · Tailwind CSS
Backend FastAPI · SQLAlchemy · SQLite
AI OpenAI Chat Completions API · gpt-4o-mini
Integration Google Calendar API · Notion OAuth/API · mailto

Data & Eligibility

사용자 입력 회의록을 안전한 범위에서 다룹니다.

데모 데이터는 사용자가 직접 입력하거나 작성한 회의록 텍스트와 파일을 사용합니다. 외부 저작권 자료나 동의 없는 개인정보 회의록은 포함하지 않으며, 학기 후에도 개인 계정 기반으로 유지 가능한 구조를 목표로 합니다.
