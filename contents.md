# AI 회의 후속 업무 자동화 서비스 (MeetAction)

회의 음성 및 텍스트를 분석하여 핵심 요약, 결정사항, 담당자별 액션 아이템을 추출하고 실제 업무 실행(Notion 생성, 메일 전송, 캘린더 등록 등)으로 연결해주는 업무 보조 웹 서비스

---

## 한 줄 요약

사용자가 회의록 텍스트를 입력하면, AI가 요약과 함께 담당자별 액션 아이템을 추출하고, Notion 회의록 생성, 후속 공유 메일 발송, Google 캘린더 일정 등록까지 한 번에 처리해주는 서비스

---

## 문제 (목표)

회의가 끝난 뒤 결정사항, 담당자 별 할 일, 마감일 등이 명확히 정리되지 않아 후속 업무가 누락되거나 지연되는 문제가 잦다. 또한, 사람이 직접 회의록을 정리하고 공유하며 일정을 등록하고 후속 공유 메일을 작성하는 데 불필요한 시간이 소요됩니다. 단순한 텍스트 요약을 넘어, 회의 내용을 분석해 실제 업무 실행(Action)과 외부 서비스 연동(Notion, 메일 발송, Google 캘린더)까지 한 번에 연결할 수 있는 자동화된 파이프라인을 구축하는 것이 목표입니다.

---

## 사용자

팀 프로젝트, 동아리, 수업 과제, 스타트업 MVP 회의처럼 회의 후 실행 항목을 정리해야 하는 사람과 소규모 팀이 사용한다. 회의록은 남기지만 후속 정리와 공유에 시간을 많이 쓰는 팀 리더, PM, 개발자, 학생 팀장을 주요 사용자로 본다.

---

## 핵심 기능

1. 회의 내용 구조화 및 요약: 회의록 텍스트를 직접 입력하거나 `.txt`/`.docx` 파일을 업로드하면 LLM이 요약, 주요 논의 내용, 결정사항, 액션 아이템, 후속 확인 사항을 추출한다.
2. 액션 아이템 관리: 추출된 액션 아이템을 담당자, 마감일, 우선순위, 상태와 함께 저장하고 칸반/목록 화면에서 검토·수정한다.
3. 외부 서비스 연동: 분석 결과를 바탕으로 후속 이메일 초안을 만들고, Notion 회의록 초안 및 Google Calendar 액션 아이템 일정으로 연결한다.

---

## 입력 → AI 처리 → 출력

회의 음성/텍스트 및 메타데이터 입력 → STT(음성 시) 변환 및 LLM 파이프라인(요약 → 결정사항 → 액션 아이템 순차 추출) 거침 → 검토 및 수정 UI 표출 → Notion 페이지 생성, 후속 메일 발송, Google 캘린더 일정 등록 완료

사용자 회의록 텍스트 또는 `.txt`/`.docx` 파일 → FastAPI 백엔드가 회의 데이터를 저장하고 LLM API(OpenAI GPT-4o-mini)에 JSON 분석 프롬프트 호출 → 요약, 참석자, 주요 논의 내용, 결정사항, 액션 아이템, 후속 확인 사항, 후속 이메일 초안 JSON 검증 →  대시보드에 분석 결과와 액션 아이템 보드 출력 + 필요 시 Notion 초안 생성, Google Calendar 일정 동기화, 이메일 초안 생성

---

## 기술 스택

- **UI**: Next.js 15 + React 19 + TypeScript + Tailwind CSS
- **백엔드**: FastAPI + SQLAlchemy + SQLite
- **모델 / API**: OpenAI Chat Completions API (`gpt-4o-mini`), Groq OpenAI-compatible API
- **외부 서비스**: Google OAuth/Login, Google Calendar API, Notion OAuth/API, `mailto:` 기반 이메일 초안, Pinecone RAG 확장 인터페이스
---

## 데이터 출처

사용자가 직접 입력하거나 업로드한 회의록 텍스트와 본인이 작성한 데모 회의록을 사용한다. 업로드 파일은 `.txt`와 `.docx` 회의록 문서이며, 저작권이 있거나 개인정보 동의가 없는 외부 회의록은 사용하지 않는다. 앱 내부 테스트 데이터는 프로젝트에서 직접 작성한 샘플 회의 내용이다.
