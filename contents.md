# AI 회의 후속 업무 자동화 서비스 (MeetAction)

회의 음성 및 텍스트를 분석하여 핵심 요약, 결정사항, 담당자별 액션 아이템을 추출하고 실제 업무 실행(Notion 생성, 메일 전송, 캘린더 등록 등)으로 연결해주는 업무 보조 웹 서비스

---

## 한 줄 요약

실무자가 회의 녹음이나 텍스트를 업로드하면, AI가 요약과 함께 담당자별 액션 아이템을 추출하고, Notion 회의록 생성, 후속 공유 메일 발송, Google 캘린더 일정 등록까지 한 번에 처리해주는 서비스

---

## 문제 (목표)

회의가 끝난 뒤 결정사항, 담당자, 마감일 등이 명확히 정리되지 않아 후속 업무가 누락되거나 지연되는 문제가 잦습니다. 또한, 사람이 직접 회의록을 정리하고 공유하며 일정을 등록하고 후속 공유 메일을 작성하는 데 불필요한 시간이 소요됩니다. 단순한 텍스트 요약을 넘어, 회의 내용을 분석해 실제 업무 실행(Action)과 협업 툴 연동(Notion, 메일 발송, Google 캘린더 일정 자동 등록)까지 한 번에 연결할 수 있는 자동화된 파이프라인을 구축하는 것이 목표입니다.

---

## 사용자

회의가 잦고 참석자 간 명확한 업무 분장 및 진행 상황 트래킹이 필수적인 프로젝트 매니저(PM), 개발 팀 리더 및 실무자

---

## 핵심 기능

1. 회의 내용 구조화 및 요약: STT로 변환된 텍스트나 입력된 회의록을 분석해 주요 안건, 결정사항, 미해결 이슈를 자동 추출합니다.
2. 액션 아이템 및 누락 정보 추출: 회의 맥락을 파악하여 담당자, 마감일, 할 일을 표 형태로 추출하고, 기한이나 담당자가 누락된 불명확한 지시사항을 감지합니다.
3. 업무 툴 연동 및 메일 전송: 추출된 결과를 바탕으로 Notion 회의록 페이지를 자동 생성하고, 후속 공유 메일을 발송하며, 도출된 마감일 및 후속 회의 일정을 Google 캘린더에 자동으로 등록합니다.

---

## 입력 → AI 처리 → 출력

회의 음성/텍스트 및 메타데이터 입력 → STT(음성 시) 변환 및 LLM 파이프라인(요약 → 결정사항 → 액션 아이템 순차 추출) 거침 → 검토 및 수정 UI 표출 → Notion 페이지 생성, 후속 메일 발송, Google 캘린더 일정 등록 완료
---

## 기술 스택

- **UI**: FastAPI + React
- **모델 / API**: Groq API (Alibaba qwen/qwen3-32b), Groq API (OpenAI whisper-large-v3)
- **외부 서비스**: Notion API, Gmail API, Google Calendar API

---

## 데이터 출처

자체 제작한 모의 회의 대본 10개 / 모의 회의 음성 파일 10개
